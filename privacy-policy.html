Privacy Policy

Last updated: June 2026

1. Overview

NIRAS AI is an educational exam authoring platform that allows users to create, manage, and export educational content such as exams, questions, and learning materials. This Privacy Policy explains how we collect, use, and protect user data.

2. Information We Collect

2.1 Account Information

  • Email address
  • Login authentication data (via Supabase secure authentication)

2.2 User Content

  • Exams created by users
  • Questions, answers, and uploaded educational files
  • Generated AI content based on user input

2.3 Technical Data

  • Device type and operating system
  • Browser type and version
  • Usage logs for performance and error monitoring

3. How We Use Information

We use collected data to:

  • Provide core app functionality (exam creation and AI generation)
  • Store and manage user-generated content
  • Improve app performance and stability
  • Provide authentication and secure access

AI question generation is powered by Groq (api.groq.com) — for AI question generation. User input sent to Groq is used solely to produce requested educational output.

4. Data Storage

  • User content is stored securely using Supabase cloud database
  • Servers are located in the United States
  • We do not store sensitive personal financial data
  • Data is retained until the user deletes it or requests removal

5. AI Processing

  • User inputs may be processed by Groq (api.groq.com) for question generation
  • Data sent to AI services is used only to generate requested educational output
  • We do not use AI input data for advertising or profiling

6. Data Sharing

We do not sell user data.

We may share limited data only with:

  • Supabase — cloud database and authentication
  • Groq (api.groq.com) — AI question generation
  • Vercel — app hosting and delivery
  • Legal authorities, if required by applicable law

7. User Control

Users can:

  • Edit or delete their content at any time
  • Request account deletion by contacting us
  • Clear local browser storage (for PWA cached data)

8. Security

We implement:

  • Secure authentication via Supabase Auth
  • HTTPS encryption for all data in transit
  • Access controls to protect user data

No system is 100% secure. We encourage users to use strong passwords.

9. Children's Privacy

NIRAS AI is intended for educational professionals and is not directed at children under the age of 13 without parental supervision. We do not knowingly collect personal data from children under 13.

10. Third-Party Services

We use the following third-party services, each with their own privacy policies:

  • Supabase — supabase.com/privacy
  • Groq — groq.com/privacy
  • Vercel — vercel.com/legal/privacy-policy

11. Changes to This Policy

We may update this policy occasionally. Updates will be posted within the app or on this page with a revised date.
